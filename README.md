üìÅ Project Structure
======================
Here is an overview of the project structure:

* **index.html**: The main entry point of the template.
* **services.html**: A page that lists all the services offered.
* **service-details.html**: A page that provides more details about a specific service.
* **css**: A folder that contains all the CSS files.
	+ **main.css**: The main CSS file that styles the template.
	+ **bootstrap.rtl.css**: The Bootstrap CSS file for right-to-left languages.
	+ **bootstrap-utilities.min.css**: The Bootstrap utilities CSS file.
* **js**: A folder that contains all the JavaScript files.
	+ **main.js**: The main JavaScript file that adds interactivity to the template.
	+ **validate.js**: A JavaScript file that validates the contact form.
	+ **aos.cjs.js**: The AOS JavaScript file.
	+ **swiper-bundle.min.js**: The Swiper JavaScript file.
* **fonts**: A folder that contains all the font files.
* **img**: A folder that contains all the image files.
